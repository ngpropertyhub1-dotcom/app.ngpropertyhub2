# app.ngpropertyhub2

Real Estate Marketplace - Lovable.dev Project

This is a synced repository from the lovable.dev platform.

## Getting Started

### Installation

```bash
bun install
# or
npm install
# or
yarn install
```

### Development

```bash
bun dev
# or
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Build

```bash
bun run build
# or
npm run build
```

## Project Structure

- `src/` - Source code
- `public/` - Static assets
- `.lovable/` - Lovable.dev configuration
- `supabase/` - Supabase configuration and migrations
